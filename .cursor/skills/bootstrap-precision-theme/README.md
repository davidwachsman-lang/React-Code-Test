# bootstrap-precision-theme

Project-level skill for **one-time setup** of the ServPro Precision theme on a branch (e.g. fresh from `main`).

## How to use

- *"Bootstrap Precision theme on this branch"*
- *"Set up theme infrastructure"*
- *"Add Precision theme"*

The agent will create ThemeContext, PrecisionTheme.css, add ThemeProvider to App, add theme toggle to Navigation, update index.css with tokens, and create/update THEME_PROGRESS.md.

**Run once** before using **improve-ui-servpro** to improve pages one by one.
